# Auto Label Image Yolov5

## Setup

Create folder 

* input
* images
* labels
* crops

Download [yolov5](https://github.com/ultralytics/yolov5)

Change file best.pt to your file

## Run

`python autoLabel.py`

## Folder

*input input images
*images Save the images
*labels Save the labels image
*crops Save the cropped image
